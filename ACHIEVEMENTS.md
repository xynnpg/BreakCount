# BreakCount Achievements

BreakCount has 100+ achievements across 13 categories. Earn XP by unlocking them and climb the rank ladder from Newcomer all the way to Eternal.

---

## Rarity Tiers and XP

| Rarity | XP | Color |
|--------|----|-------|
| Bronze | 25 XP | `#CD7F32` |
| Silver | 75 XP | `#9E9E9E` |
| Gold | 200 XP | `#FFB300` |
| Platinum | 500 XP | `#4FC3F7` |
| Secret | 750 XP | `#A89888` |

Secret achievements show "???" as their description until you unlock them.
